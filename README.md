

# DISCSTACK Backend

Discstack is a web app for organizing and saving your favorite albums

### [DISCSTACK Frontend](https://github.com/jackaubone/discstack_frontend.git)

## Installation

Use Rails for installation and clone with github

```bash
rails db:create 
rails db:seed 
rails db:migrate
rails server
```
